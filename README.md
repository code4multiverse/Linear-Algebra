### `numpy` 與 `sympy` 之間的分別

- **numpy**：
  - 主要用於「數值計算」，專為高速處理大量數值資料、浮點數矩陣運算、線性代數計算等而設計。
  - 適合用於資料分析、科學計算、機器學習等需快速處理大量數據的場景。
  - 所有運算皆以近似值（float/double 型態）進行，無法維持精確的數學符號運算。
  - 例如：`numpy.linalg.inv()` 用於計算矩陣的反矩陣（數值解）。

- **sympy**：
  - 主要用於「符號運算」，可進行代數化簡、展開、微分、積分、解方程式等精確的數學推導。
  - 適合用於需要數學式子推導、顯示精確分數、代數符號計算的場景。
  - 能直接操作字母、符號，以及維持解析表達式，如解聯立方程、化簡或展開表達式等。
  - 例如：`sympy.Matrix().inv()` 用於計算矩陣的反矩陣（符號解）。

**總結：**  
- 用 `numpy` 處理大量數值與科學運算；  
- 用 `sympy` 處理精確的數學符號與代數推導。
